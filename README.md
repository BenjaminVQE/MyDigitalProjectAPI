# MyDigitalProjectAPI

Cette application est un projet d'école My DigitalSchool, pour l'entreprise Tissup.

## Installation

### Conditions

-   Cloner le projet
-   Initialiser les variables dans le .env
-   Installer Docker
-   Installer Makefile
-   Se mettre dans le répertoire de l'application
-   Lancer la commande "make install"
